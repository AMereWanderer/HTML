There are two types of elements:-
1. Inline Block Elements.
2. Block Level Elements (B.L.E).

  • Paragraph tag is a B.L.E.
  • Span Tag is an I.B.E.
  • B.L.E takes full width of the viewport if not assigned otherwise.
